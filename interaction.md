# 多公司财务管理系统 - 交互设计

## 系统概述
一个支持多公司管理的现代化财务系统，专为菲律宾市场设计，支持修车行、汽车改装和空调服务等业务类型。系统使用菲律宾比索(PHP)作为默认货币，提供完整的多公司财务管理解决方案。

## 核心交互功能

### 1. 多公司管理
- **公司切换器**: 顶部导航栏的公司选择下拉菜单，支持快速切换不同公司
- **公司仪表板**: 每个公司独立的财务概览和关键指标
- **公司设置**: 为每个公司设置不同的业务类型、税率和财务参数
- **权限管理**: 不同用户对不同公司的访问权限控制

### 2. 业务类型管理
- **修车行业务**: 
  - 维修工单管理（零件费、人工费、服务费）
  - 客户车辆档案管理
  - 供应商管理（零件采购）
  - 库存管理（零件库存）
- **汽车改装业务**:
  - 改装项目管理（改装套件、定制费用）
  - 客户定制需求跟踪
  - 改装配件采购管理
- **空调服务业务**:
  - 安装/维修服务工单
  - 设备销售管理
  - 维保合同管理

### 3. 实时财务仪表板
- **多公司收入支出概览**: 动态显示选中公司的财务数据
- **业务类型分析**: 按修车、改装、空调等业务类型分析收入构成
- **现金流图表**: 交互式现金流趋势图，支持公司和时间段筛选
- **关键指标卡片**: 显示净利润、现金流、预算执行率等核心KPI

### 4. 交易记录管理器
- **多公司交易记录**: 支持按公司筛选交易记录
- **业务类型分类**: 自动识别业务类型并分类（修车、改装、空调服务等）
- **客户管理**: 完整的客户档案和交易历史
- **供应商管理**: 供应商信息和采购记录
- **库存跟踪**: 零件和材料的库存管理

### 5. 预算规划工具
- **多公司预算设置**: 为不同公司设置独立的预算
- **业务类型预算**: 按业务类型设置预算分配
- **项目预算**: 具体项目（如改装项目）的预算控制
- **部门预算**: 不同部门（维修部、改装部、空调部）的预算管理

### 6. 财务报表生成器
- **多公司报表**: 支持单公司和合并报表
- **业务类型分析报表**: 各业务类型的盈利能力分析
- **客户分析报表**: 客户贡献度分析
- **供应商分析报表**: 供应商采购分析

## 交互流程设计

### 主仪表板交互
1. 用户进入系统后选择要管理的公司
2. 系统显示选中公司的实时财务概览
3. 用户可以通过顶部切换器快速切换公司
4. 点击图表可深入查看详细数据
5. 通过业务类型筛选器查看特定业务数据

### 多公司交易管理交互
1. 选择要操作的公司
2. 添加交易时选择业务类型（修车/改装/空调）
3. 系统自动分类并更新相关库存和成本
4. 支持跨公司交易查看和对比
5. 客户档案包含多公司交易历史

### 业务类型预算管理交互
1. 为每个公司设置不同的预算模板
2. 按业务类型分配预算额度
3. 实时跟踪各业务类型的预算执行情况
4. 超预算时显示业务类型特定的预警信息
5. 支持业务类型间的预算调整

### 多公司报表生成交互
1. 选择要生成报表的公司（可多选）
2. 选择报表类型（合并报表或单独报表）
3. 选择业务类型筛选条件
4. 实时预览生成的报表
5. 支持导出PDF、Excel格式

## 菲律宾本地化特色
- **货币单位**: 菲律宾比索(PHP)作为默认货币
- **税务支持**: 支持菲律宾税制和发票管理
- **本地业务**: 针对菲律宾汽车服务行业的特殊需求
- **多语言**: 支持英语和他加禄语切换
- **本地报表**: 符合菲律宾财务规范的报表格式

## 用户体验特色
- **公司切换**: 一键切换不同公司的数据和界面
- **业务智能**: 基于业务类型的智能分类和分析
- **实时同步**: 多公司数据的实时同步和更新
- **移动优化**: 适配移动设备的多公司管理
- **数据安全**: 多公司数据的隔离和权限控制
